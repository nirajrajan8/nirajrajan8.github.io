Maven Build Tool

An introduction to Maven for project management and dependency handling.

Project Structure

Maven

Maven enforces a standard directory layout to simplify build processes across different projects.

Standard Directory Layout
├── src/
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   └── test/
│       ├── java/
│       └── resources/
└── pom.xml

pom.xml

Maven

The Project Object Model file (`pom.xml`) is the core of a Maven project, containing configuration and dependencies.

Example `pom.xml`
<project>
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.example.automation</groupId>
    <artifactId>my-automation-project</artifactId>
    <version>1.0-SNAPSHOT</version>

    <dependencies>
        <!-- TestNG Dependency -->
        <dependency>
            <groupId>org.testng</groupId>
            <artifactId>testng</artifactId>
            <version>7.7.1</version>
            <scope>test</scope>
        </dependency>

        <!-- Rest Assured Dependency -->
        <dependency>
            <groupId>io.rest-assured</groupId>
            <artifactId>rest-assured</artifactId>
            <version>5.3.0</version>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <!-- Compiler Plugin -->
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>3.10.1</version>
            </plugin>
        </plugins>
    </build>
</project>

Rest Assured API Testing

A simple, powerful, and fluent Java library for testing REST APIs.

Fluent Interface

Rest Assured

Rest Assured uses a BDD (Behavior-Driven Development) style syntax (`given-when-then`) to make API tests readable and easy to write.

Example Syntax
given().
        // Setup steps (headers, body, etc.)
when().
        // HTTP method (GET, POST, etc.)
then().
        // Validation steps (status code, body content)

Simple GET Request

Rest Assured

A basic example of a GET request with status code and body validation.

GET Request Test
import static io.restassured.RestAssured.*;
import static org.hamcrest.Matchers.*;

import org.testng.annotations.Test;

public class ApiTests {

    @Test
    public void testGetStatusCode() {
        given()
                .when()
                .get("https://reqres.in/api/users/2")
                .then()
                .statusCode(200);
    }

    @Test
    public void testGetResponseBody() {
        given()
                .when()
                .get("https://reqres.in/api/users/2")
                .then()
                .statusCode(200)
                .body("data.first_name", equalTo("Janet"));
    }
}
